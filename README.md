camber-agent
============

[![Build Status](https://travis-ci.org/camber/agent.png?branch=master)](https://travis-ci.org/camber/agent)


## Usage

```javascript
var agent = new Agent;
agent.browser_name
// => "chrome"
agent.browser_version
// => "30.0.1599.69"
agent.os
// => "OS X 10.8"
agent.platform
// => "Apple Mac"
```
